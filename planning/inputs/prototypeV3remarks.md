# Prototype V3 remarks

Number one rule of the site is that no ranking or scoring data should be visible to the user or any movie they themselves have not yet rated! Number one rule.

Second, the expected content of each page is outlined explicitly in the `pagesDraft` document. Very little of that content appears in the prototype as shown.

Thirdly, the scorecard drag and drop does not function as expected. Attempting to move any of the categories results in that category. Dropping to the bottom of the drag section. What should be happening is that each category should be distinctly separated and when a category is dragged to a position between two others, it should be placed in that position in order.

I did not see anything of a prototypical layout of any of the content that I was looking to see for any of the pages that I was looking to see, and navigation between pages was non-existent.

Keep the colors, fonts and themes from the original prototype. In fact, let's keep the header as well.
