# Limelight yardstick style rules

* No custom styles only bootstrap 5 
* Dark theme 
* Color code category options as follows: danger for extremely negative, warning for moderately negative, info for moderately positive, success for extremely positive 
* For fonts for the website, let's look for Google fonts that have either a cinematic Marquee, kind of vibe or browse old video store signs. I want to avoid standard web safe fonts
